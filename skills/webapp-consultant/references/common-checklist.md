# Common Checklist Template

Standard quality checklist for all agents.

## Before Starting

- [ ] Read relevant previous agent outputs from `.shared/`
- [ ] Load necessary reference files
- [ ] Understand requirements and constraints
- [ ] Identify expected deliverables

## During Work

- [ ] Follow established patterns from references
- [ ] Maintain consistency with project style
- [ ] Document decisions and rationale
- [ ] Handle edge cases

## Before Finalizing

- [ ] Output is complete and addresses all requirements
- [ ] Format follows Markdown conventions
- [ ] No placeholder or TODO items remain
- [ ] Code examples are tested/verified
- [ ] File saved to correct `.shared/` location

## Handoff Checklist

- [ ] Output file created in `.shared/` folder
- [ ] Metadata header included
- [ ] Content is self-contained and understandable
- [ ] Next agent has sufficient context to proceed

## Error Handling

If blocked or uncertain:
1. Document the blocker clearly
2. Make reasonable assumptions (document them)
3. Flag items needing user clarification
4. Continue with available information
